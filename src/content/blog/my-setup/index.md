A curated list of the hardware and software I use daily for development, infrastructure, and productivity.

## Hardware

### Laptop
Primary development machine. Configured for coding, daily productivity tasks, and professional workflows.

### External Monitor
High-resolution display for extended screen real estate. Perfect for split-pane workflows and design work.

### Mechanical Keyboard
Custom mechanical keyboard with tactile switches. Hot-swappable, wireless, and comfortable for long typing sessions.

### Ergonomic Mouse
Wireless ergonomic mouse with customizable buttons. Silent clicks and seamless multi-device switching.

### Ergonomic Chair
Adjustable office chair designed for long work sessions. Essential for maintaining good posture during extended coding.

### Network Setup
Home network infrastructure with managed switch, access points, and firewall for secure, reliable connectivity.

## Software

### VS Code
Primary code editor with extensions for linting, formatting, and language support. Custom keybindings and themes for productivity.

### Terminal + Shell
Custom terminal setup with a modern shell, prompt theme, and aliases for rapid development workflows.

### Git
Version control for all projects. Custom commit conventions, branch strategies, and automated CI/CD pipelines.

### Database GUI
Visual database management tool for querying, browsing, and managing SQL and NoSQL databases during development.

### Figma
Design tool for prototyping UI components, reviewing mockups, and collaborating with design teams.

### Neovim
Lightweight editor for quick terminal-based edits. Custom config with LSP, fuzzy finder, and syntax highlighting.

### Password Manager
Secure credential storage for API tokens, SSH keys, and shared team vaults. Essential for maintaining security hygiene.

### Browser
Chromium-based browser with workspaces for separating personal and professional profiles. Tab management and extensions for dev tools.

### Note-Taking App
Second brain for notes, documentation, project planning, and runbooks. Templates for meeting notes and technical documentation.
